# 📊 Sales Intelligence 360°

> Dashboard Executivo desenvolvido em Power BI para análise estratégica de vendas.

---

## 📌 Objetivo

Este projeto simula um cenário real de Business Intelligence para uma empresa fictícia do setor comercial.

O objetivo é transformar dados de vendas em informações estratégicas por meio de um dashboard executivo desenvolvido no Power BI.

---

## 🛠 Tecnologias

- Power BI
- Power Query
- DAX
- SQL
- Excel
- GitHub

---

## 📈 Indicadores

- Receita Total
- Lucro Bruto
- Ticket Médio
- Pedidos
- Taxa de Conversão
- Receita x Meta
- Novos Clientes
- Top Produtos
- Ranking de Vendedores
- Receita por Estado
